# Dealership DAO

A vehicle dealership management application developed in Java using MySQL and the DAO (Data Access Object) design pattern.

## Overview

This project simulates the management system of a vehicle dealership. It allows the handling of customers, vehicles, employees and sales operations while maintaining data persistence through a relational database.

The application follows the DAO design pattern to separate business logic from database access, improving maintainability and scalability.

## Features

- Customer management
- Vehicle management
- Employee management
- Database connectivity
- CRUD operations
- Data persistence using MySQL
- DAO pattern implementation
